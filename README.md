## 📝 Descripción de la landing page
La página web es una **landing page corporativa** diseñada para una agencia audiovisual ficticia.  
Su objetivo es presentar los servicios de la empresa, mostrar un portafolio, destacar indicadores relevantes, recopilar comentarios de clientes y permitir el registro de nuevos interesados.  
El diseño sigue un enfoque **mobile first**, adaptándose a diferentes dispositivos (teléfono, tableta y computadora).

## 💻 Tecnologías utilizadas
- **HTML5** para la estructura del contenido.  
- **CSS3** con enfoque mobile first y consultas de medios para la adaptación responsive.  
- **Bootstrap 5** para el sistema de grillas, componentes (cards, acordeón, carrusel, botones, tablas) y utilidades de espaciado y alineación.  
- **Bootstrap Icons** para iconografía de redes sociales y secciones.  
- **GitHub Pages** para la publicación en línea.
